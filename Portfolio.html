Mon Portfolio Web

Afin d'illustrer de manière plus visuelle et de présenter avec plus de détail mes divers projets, j'ai décidé de développer moi même mon portfolio en ligne. Pour cela, j'ai choisis de l'héberger sur GitHub page, qui est un outil que j'avais deja utilisé durant mes cours de NSI. J'ai utilisé les notions en HTML et CSS du programme de première et Terminale, en apportant du Java Script pour les Sliders par exemple.



Ce portfolio a été entièrement rédigé par mes soins, mais j'ai pris la liberté d'utiliser l'Intelligence Artificielle de Google, Gemini, afin de m'aider sur les techniques de JavaScript et de CSS que je ne maitrisait pas forcément. Mais, afin de garder une certaine éthique, j'ai tenu a comprendre ce que faisais chaque morceau de code proposé, y compris la programmation des vagues en SVG de la page d'ouverture.



Pour la direction artistique, j'ai choisis un design épuré et complétement noir et blanc, pour un rendu moderne et minimaliste, qui sur un écran avec des noir profond comme les OLED, qui sont de plus en plus courant sur les smartphones, rend particulierement bien. J'ai choisi en couleur d'accuentation un bleu clair plutôt futuriste, qui se décline très bien dans les version floues et texte.
J'ai desginé le logo dans le même état d'esprit: Blanc, épuré et Futuriste. Les courbes du bas forment mes initiales (un A et un M), tandis que le motif sur la partie supérieure du cercle est purement décorative, symbolisant une sorte de couronne, mais sans réelle signification, j'ai choisis ce motif pour la pure ésthétique visuelle.
Les polices Nullshock et Conthrax que l'on voit dans les titres et les paragraphes ont été choisies pour leur courbe esthétique qui s'allignait bien sur le Logo. J'ai construit tout le site autour de cet DA a la fois ronde et abrupte. Ma couleur de texte grise est la pour apporter un entre deux entre le noir total du fond et le blanc pur du texte.
